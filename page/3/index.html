a 在光标后输入

s 删除光标的字符并进入输入模式
x 删除当前光标下的字符
d 数字 放向键 自定义在什么方向删除多少个字符/行
dd 删除整行
p 粘贴
y 数字 方向键 自定义在什么方向复制多少个字符/行
w 进入下一个单词的开头
c 删除当前光标的字符
c 数字 方向键 自定义在什么方向删除多少个字符/行
c w 删除整个单词
c i w 删除你光标所在的单词
c i 字符 删除自定义内部的所有内容
d i 字符 删除（复制）自定义内部的所有内容
y i 字符 复制自定义内部的所有内容
f 查找模式
f 字符 查找自定义字符，并到指定位置
d f 字符 删除（复制）到自定义的字符位置
y f 字符 复制到自定义的字符位置
