安装 imagemagick

```bash
brew install imagemagick
```

安装 exiftool

```bash
brew install exiftool
```

重新生成图

```bash
magick input.jpg -resize 95% -resize 105.26% output.jpg
magick input.jpg temp.png
magick temp.png output.jpg
magick input.jpg -blur 0x0.5 output.jpg
magick input.jpg -noise Reduce output.jpg
magick input.jpg -quality 85 output.jpg
magick input.jpg -strip -quality 100 output.jpg
```

查看修改图的元数据

```bash
exiftool output.jpg
```

或者重新编码

```bash
ffmpeg -i input.jpg output.jpg
```
